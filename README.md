# task_note_front
任务管理器

## 功能特性

这是一个简单而强大的任务管理器，具有以下功能：

- ✅ **每日任务介绍**：为每个任务添加详细的介绍说明
- 📊 **实现情况追踪**：支持四种状态（未开始、进行中、已完成、已延期）
- 📝 **附加笔记**：为任务添加额外的备注和说明
- 🔍 **状态过滤**：按任务状态快速筛选任务列表
- 💾 **本地存储**：自动保存任务到浏览器本地存储
- 📱 **响应式设计**：支持桌面和移动设备

## 使用方法

### 在线使用

1. 直接在浏览器中打开 `index.html` 文件即可使用

### 本地部署

如果需要通过 HTTP 服务器访问，可以使用以下方法：

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Python 2
python -m SimpleHTTPServer 8000

# 使用 Node.js (需要安装 http-server)
npx http-server
```

然后在浏览器中访问 `http://localhost:8000`

## 功能说明

### 添加任务

1. 填写任务标题（必填）
2. 填写任务介绍（必填）
3. 选择实现情况（未开始、进行中、已完成、已延期）
4. 添加附加笔记（可选）
5. 点击"添加任务"按钮

### 管理任务

- **编辑任务**：点击任务卡片上的"编辑"按钮，内容会填充到表单中，修改后重新提交
- **删除任务**：点击任务卡片上的"删除"按钮
- **更新状态**：直接在任务卡片中选择新的实现情况
- **筛选任务**：点击顶部的状态按钮筛选不同状态的任务

### 数据存储

所有任务数据都保存在浏览器的本地存储（LocalStorage）中，即使关闭浏览器，数据也不会丢失。
